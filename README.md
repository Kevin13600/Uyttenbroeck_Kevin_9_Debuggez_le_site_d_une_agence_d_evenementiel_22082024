# 724events - Debug d'application React

## Description
Application React one-page pour une agence événementielle, nécessitant debug et tests.

## Prérequis
- Node.js ≥ 16.14.2 LTS
- npm ≥ 8.5.5
- yarn

## Installation
```bash
# Installation des dépendances
yarn install

# Lancement du serveur de développement
yarn start

# Exécution des tests
yarn test --watch
```
